# 中藥心靈療癒卡 TCM Oracle Cards

> 探索古老智慧，療癒現代心靈

一款結合傳統中醫藥理與現代心靈療癒的線上抽牌應用程式。

![Preview](images/牌背.png)

## ✨ 功能特色

- 🎴 **32張療癒卡片** - 涵蓋收斂、溫熱、寒涼、補益、疏散五大類別
- 🎯 **三種意圖選擇** - 情緒釋放、身體療癒、靈性成長
- 🔮 **個人化解讀** - 依據選擇的意圖生成專屬療癒指引
- ✨ **精美互動設計** - 3D翻牌動畫、粒子背景效果
- 📱 **響應式設計** - 完美支援手機與桌面瀏覽

## 🛠️ 本地運行

1. Clone 此專案
```bash
git clone https://github.com/你的用戶名/tcm-oracle-cards.git
```

2. 直接開啟 `index.html` 或使用本地伺服器
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve
```

3. 在瀏覽器開啟 `http://localhost:8000`

## 📜 使用說明

1. **開始抽牌** - 點擊首頁的「開始抽牌」按鈕
2. **選擇意圖** - 靜心選擇此刻最需要的療癒方向
3. **翻開牌卡** - 點擊牌卡，觀看翻轉動畫
4. **接收指引** - 閱讀草藥的藥性、適用情境與療癒語句
5. **解讀整合** - 依據你的意圖，獲得個人化的深度解讀
